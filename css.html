How I Used CSS

I used CSS to improve the appearance of my website beyond the default Bootstrap styling. My custom CSS helped make the site look cleaner, more modern, and more visually consistent.

  • I changed the background image and sizing for the hero section on the home page.
  • I added custom colors for headings and page titles.
  • I styled the cards with hover effects to make them more interactive.
  • I adjusted spacing, padding, and margins to improve readability.
  • I rounded images and added shadows to make the pages look more polished.

Bootstrap was used for layout, navigation bars, buttons, and responsive design, while my custom CSS was used to personalize the website and make it look more unique.
